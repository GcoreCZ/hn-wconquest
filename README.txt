HANDYNOTES WARCRAFT CONQUEST 

This addon requires HandyNotes to run. 
It provides in-game map locations for on-going campaigns run by Warcraft Conquest of Moon Guard US server.
See CREDITS.TXT for thanks and help.

MANUAL INSTALLATION

    1. Extract the zip.
    2. Copy and paste the folder into your WoW retail installation folder\interface\addons.

    Requires HandyNotes addon.
